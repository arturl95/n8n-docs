for(let i=0; i < items.length; i++) {
  bookedSum = bookedSum + items[i].json.orderPrice;
}
return [{json:{totalBooked, bookedSum}}];
```

Notice the format in which we return the results of the calculation:<br>
`return [{json:{totalBooked, bookedSum}}]`
