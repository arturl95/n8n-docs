## Operations

* Organization
    * Retrieve data on the logged-in user's organization.
* Space
    * Retrieve data on all the spaces in the logged-in user's organization.
* User
    * Retrieve data on all the users in the logged-in user's organization.
* Work Item
    * Create
    * Get
    * Get All
    * Update
