## Operations

* User
    * Create or update a user profile
    * Change a users identifier
    * Unsubscribe a user.
    * Resubscribe a user.
    * Delete a user.
    * Adds a tag to a users profile.
    * Removes a tag from a users profile.
* Event
    * Track an event for a specific customer
