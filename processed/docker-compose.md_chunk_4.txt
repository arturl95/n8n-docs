sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io
```

### 2. Optional: Non-root user access

Run when logged in as the user that should also be allowed to run docker:

```bash
sudo usermod -aG docker ${USER}
su - ${USER}
```
